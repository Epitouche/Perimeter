## LUSTRE

A framework for building Web apps in Gleam!

### install lustre

in a gleam project

```bash
gleam add lustre
gleam add --dev lustre_dev_tools
```

### run lustre

```bash
gleam run -m lustre/dev start
```

### documentation lustre

[lustre documentation](https://hexdocs.pm/lustre/)
